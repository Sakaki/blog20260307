# reduce_hero_navbar_gap 設計書

## 概要

トップページの NavBar と SAKAKI333 アスキーアート間の余白を縮小する。

## 変更箇所

| ファイル                | 変更内容                                                 |
| ----------------------- | -------------------------------------------------------- |
| `src/styles/global.css` | `.bio-section` の `padding-top` を `20px` → `4px` に変更 |

## 隙間の内訳（変更前 → 変更後）

| 要素                  | プロパティ           | 変更前   | 変更後           |
| --------------------- | -------------------- | -------- | ---------------- |
| `<main>` (BaseLayout) | `py-8` (padding-top) | 32px     | 32px（変更なし） |
| `.bio-section`        | `padding-top`        | 20px     | 4px              |
| **合計**              |                      | **52px** | **36px**         |

## テスト方針

- ブラウザでトップページを表示し、NavBar と ASCII アートの間隔が適切か目視確認
- モバイル表示でも確認
- 他のページ（`/posts`, `/posts/[slug]`, `/tags`）に影響がないことを確認
