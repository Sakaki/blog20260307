# 0015. トップページ Hero/NavBar 間の隙間縮小

Date: 2026-03-19
Status: Accepted

## Context

トップページの SAKAKI333 アスキーアート (BioSection) と NavBar の間に過剰な余白がある。
現状、`<main>` の `py-8` (32px) と `.bio-section` の `padding-top: 20px` が合算され、
合計 52px の隙間が生じている。ファーストビューの情報密度を高めるため、この隙間を縮小する。

## Decision

- `<main>` の `py-8` はサイト共通のレイアウト余白であり、全ページに影響するため変更しない
- `.bio-section` の `padding-top` を `20px` から `4px` に縮小する（合計 36px に削減）
- BioSection はトップページ専用コンポーネントのため、影響範囲が限定的

### 採用しなかった選択肢

1. `<main>` の `py-8` を削減: 全ページに影響するため却下
2. BioSection に負のマージンを適用: ハック的でメンテナンス性が低いため却下

## Consequences

### 良い点

- ファーストビューでより多くの情報が見える
- 変更箇所が CSS 1行のみで影響範囲が最小

### トレードオフ

- なし（トップページ専用コンポーネントの微調整のため）
