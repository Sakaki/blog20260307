# 0014. 記事メタ情報（日付・読了時間）を2行表示に変更

Date: 2026-03-19
Status: Accepted

## Context

記事ページ（PostLayout.astro）のヘッダー部分で、日付と読了時間（reading time）が同一の `<p>` タグ内に横並びで表示されている。情報の視認性を向上させるため、それぞれを独立した行に分けて表示したい。

## Decision

PostLayout.astro のメタ情報表示部分を変更し、日付と読了時間を縦に並べる。

- 既存の `<p>` タグ内の横並びレイアウト（`ml-4` でスペーシング）を、`flex flex-col` による縦並びに変更する
- 各情報は独立した `<span>` として表示する
- クリーンアーキテクチャのレイヤー分類は不要（純粋なUI調整のため）

### 採用しなかった選択肢

1. `<br>` タグによる改行: セマンティックでなく、スタイル調整が難しい
2. 別々の `<p>` タグに分離: マージン制御が煩雑になる

## Consequences

### 良い点

- 日付と読了時間の視認性が向上する
- 変更箇所が1ファイルのみで影響範囲が限定的

### トレードオフ

- 縦方向のスペースを若干多く使用する
