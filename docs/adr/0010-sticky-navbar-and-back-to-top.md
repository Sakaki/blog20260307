# 0010. スティッキーNavBar & トップに戻るボタン

Date: 2026-03-17
Status: Accepted

## Context

スクロール時にナビゲーションバーが画面外に消えてしまい、ナビゲーションへのアクセスが不便。
長い記事を読んだ後にページトップへ戻る手段もない。

## Decision

### スティッキーNavBar

- `position: sticky` で画面上部に固定
- スクロール時（閾値: 80px）にコンパクトモードへ遷移
  - 縦幅を縮小（padding 削減）
  - サイトタイトル `~/sakaki333.dev` → ページタイトルに切り替え
- CSS transition でスムーズに切り替え

### トップに戻るボタン

- 右下に固定配置のフローティングボタン
- スクロール量が 300px を超えたら表示
- クリックで `scrollTo({ top: 0, behavior: 'smooth' })` でトップへ戻る
- ターミナルテーマに合わせたデザイン

### 採用しなかった選択肢

1. `position: fixed` NavBar: コンテンツ領域の計算が複雑になる
2. アニメーションライブラリ: JS バンドルが増える。CSS transition + バニラ JS で十分

## Consequences

### 良い点

- ナビゲーションが常にアクセス可能
- 長い記事でもトップへ即座に戻れる
- スクロール時にページタイトルが表示され現在位置が分かりやすい

### トレードオフ

- クライアント JS が増える（scroll イベントリスナー）
