# deploy

ブログをビルドして自宅サーバーの Nginx に配置するワークフロー。

## デプロイ先

- **ホスト**: 192.168.20.30
- **ユーザー**: root
- **コンテンツルート**: /var/www/chiya/
- **秘密鍵**: ~/.ssh/id_rsa

## ステップ 1: ビルド

```bash
npm run build
```

ビルドが失敗した場合はエラー内容をユーザーに報告し、デプロイを中断する。

## ステップ 2: デプロイ

ビルド成功後、tar + SSH でサーバーに転送する。
サーバー上の旧ファイルをすべて削除してから展開する。

```bash
tar -C dist -czf - . | ssh -i ~/.ssh/id_rsa root@192.168.20.30 "rm -rf /var/www/chiya/* && cd /var/www/chiya && tar --no-same-owner -xzf - && echo 'Deploy complete'"
```

注意: Windows 環境では rsync が使えないため tar + SSH を使用する。

SSH 接続に失敗した場合はエラー内容をユーザーに報告して終了する。

## ステップ 3: 完了報告

デプロイ完了後、ビルドされたページ数とデプロイ先を簡潔に報告する。
